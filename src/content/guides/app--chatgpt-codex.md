# 앱 만들기 — ChatGPT Codex

이 가이드는 아직 맞춤 가이드가 준비되지 않은 조합을 위한 기본 안내예요.

---

# ChatGPT Codex 앱 설치

1. [chatgpt.com/codex](https://chatgpt.com/codex) 에서 내 OS에 맞는 앱 다운로드 및 설치
2. 앱 실행 → ChatGPT 계정으로 로그인 (무료 플랜도 사용 가능)

---

# 플랫폼별 개발 도구 설치

앱을 실제로 빌드하고 실행하려면 플랫폼별 개발 도구가 필요해요.

| 플랫폼 | 필요한 도구 | OS 조건 |
|---|---|---|
| **Android** | [Android Studio](https://developer.android.com/studio) | Windows, macOS, Linux |
| **iOS / iPadOS** | [Xcode](https://apps.apple.com/us/app/xcode/id497799835?l=ko&mt=12) (Mac App Store) | **macOS만 가능** |
| **macOS 앱** | [Xcode](https://apps.apple.com/us/app/xcode/id497799835?l=ko&mt=12) (Mac App Store) | macOS만 가능 |
| **Windows 앱** | [Visual Studio Community](https://visualstudio.microsoft.com) | Windows 권장 |

---

# Codex로 코드 작성

개발 도구에서 프로젝트를 먼저 생성한 뒤, Codex 앱에서 **New Task** → 프로젝트 폴더를 열고 만들고 싶은 기능을 요청하세요.

에러가 나면 에러 메시지를 그대로 Codex 채팅에 붙여넣으면 수정해줘요.
