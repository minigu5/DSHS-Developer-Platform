# Claude Code로 바이브 코딩 시작하기 (VS Code, 터미널 없이)

터미널 없이도 Claude Code를 쓸 수 있어요!
VS Code에 Claude Code 확장 프로그램을 설치하면 버튼 클릭만으로 모든 걸 할 수 있습니다.

---

# 1단계: VS Code 설치

1. [code.visualstudio.com](https://code.visualstudio.com) 에 접속
2. 내 운영체제에 맞는 버전 다운로드 및 설치
3. VS Code 실행

---

# 2단계: Claude Code 확장 프로그램 설치

1. VS Code 왼쪽 사이드바에서 **확장 프로그램** 아이콘 클릭 (또는 `Ctrl+Shift+X` / `Cmd+Shift+X`)
2. 검색창에 **"Claude Code"** 입력
3. Anthropic이 만든 **Claude Code** 확장 프로그램 → **Install** 클릭

설치가 완료되면 왼쪽 사이드바에 Claude 아이콘이 생깁니다.

---

# 3단계: 로그인

1. 사이드바의 **Claude 아이콘** 클릭
2. **Sign in** 버튼 클릭 → 브라우저가 열림
3. Anthropic 계정으로 로그인 (Claude.ai Pro / Max 구독 계정 사용 가능)
4. 인증이 완료되면 VS Code로 돌아와요

> Claude.ai 계정이 없다면 [claude.ai](https://claude.ai) 에서 무료로 가입할 수 있어요.

---

# 4단계: 프로젝트 폴더 열기

1. VS Code에서 `파일 → 폴더 열기` (또는 `Ctrl+K Ctrl+O` / `Cmd+K Cmd+O`)
2. 새 폴더를 만들거나 기존 폴더 선택
3. 폴더가 열리면 왼쪽 파일 탐색기에서 내 프로젝트가 보여요

---

# 5단계: Claude에게 만들어달라고 요청하기

1. 사이드바에서 **Claude 아이콘** 클릭 → 채팅창 열기
2. 만들고 싶은 것을 한국어로 설명하세요:

> "오늘 할 일을 입력하고 체크할 수 있는 투두리스트 웹페이지를 만들어줘. 귀엽게 디자인해줘."

Claude가 파일을 자동으로 생성합니다. **Accept** 버튼을 누르면 실제 파일에 반영돼요.

***

**바이브 코딩 핵심 요령:**
- **에러가 나면** → 에러 메시지를 Claude 채팅창에 붙여넣기
- **수정하고 싶으면** → "배경색을 파란색으로 바꿔줘"처럼 이어서 요청
- **코드 설명이 필요하면** → 궁금한 코드를 드래그하고 "이게 뭐야?"라고 물어보기
- **인라인 수정** → 코드 위에서 `Ctrl+K` (또는 `Cmd+K`) 를 눌러 그 자리에서 바로 수정 요청
