# 앱 만들기 — Gemini (Google Antigravity)

Google Antigravity는 Gemini가 내장된 AI 코딩 IDE예요.
앱을 만들려면 플랫폼별 개발 도구도 함께 필요합니다.

---

# 1단계: Google Antigravity 설치

1. [antigravity.google/download](https://antigravity.google/download) 에서 내 OS에 맞는 버전 다운로드 및 설치
2. 실행 후 Google 계정으로 로그인 (성인 인증 불필요)

---

# 2단계: 플랫폼별 개발 도구 설치

만들고 싶은 앱의 플랫폼에 따라 추가 도구가 필요해요.

| 플랫폼 | 필요한 도구 | 조건 |
|---|---|---|
| **iPadOS** | Xcode (Mac App Store) | macOS 전용 |
| **macOS 앱** | Xcode (Mac App Store) | macOS 전용 |
| **Windows 앱** | Visual Studio Community | Windows 권장 |

설치 후 해당 도구에서 새 프로젝트를 생성하세요.

---

# 3단계: Antigravity에서 프로젝트 열기

1. Antigravity에서 **Open Folder** → 방금 만든 프로젝트 폴더 선택
2. Gemini 채팅창에 기능을 요청하세요:

> "이 iOS/iPadOS 앱에 노트 작성 기능을 추가해줘. 목록 화면과 편집 화면으로 구성해줘."

Gemini가 파일을 수정하면 **Accept** 로 적용하고, 개발 도구에서 ▶ 빌드해 확인하세요.

---

# 4단계: 기능 추가하며 완성하기

> "다크 모드를 지원하게 해줘"  
> "데이터를 앱을 껐다 켜도 저장되게 해줘"

에러가 나면 개발 도구의 오류 메시지를 Gemini 채팅창에 붙여넣으면 수정해줘요.
