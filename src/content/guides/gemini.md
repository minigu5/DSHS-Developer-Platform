# Google Antigravity로 바이브 코딩 시작하기

Google Antigravity는 Google이 2025년 공개한 AI 에이전트 기반 코딩 IDE예요.
VS Code를 기반으로 만들어졌고, Gemini가 코드를 직접 작성·수정해줍니다.
macOS, Windows, Linux 모두 지원하며, 기본 사용은 **무료**예요.

---

# 1단계: Google Antigravity 다운로드 및 설치

1. [antigravity.google/download](https://antigravity.google/download) 에 접속
2. 내 운영체제에 맞는 버전 다운로드
3. 설치 파일 실행 → 안내에 따라 설치 완료

---

# 2단계: Google 계정으로 로그인

Antigravity를 실행하면 로그인 화면이 나와요.

1. **Sign in with Google** 클릭
2. 구글 계정으로 로그인 (학교 계정도 사용 가능)
3. 로그인 완료 후 메인 화면으로 진입

무료 플랜에서도 Gemini Pro를 넉넉하게 사용할 수 있어요.

---

# 3단계: 새 프로젝트 시작

Antigravity 메인 화면에서:

1. **New Project** 클릭 (또는 `Ctrl+Shift+N` / `Cmd+Shift+N`)
2. 프로젝트 폴더 위치 선택
3. Gemini 채팅창에 만들고 싶은 것을 설명하세요:

> "오늘 할 일을 입력하고 체크할 수 있는 투두리스트 웹페이지를 만들어줘. 귀엽게 디자인해줘."

Gemini가 파일을 생성하고 코드를 작성합니다. 변경 사항을 **Accept** 버튼으로 적용하세요.

---

# 4단계: 결과 확인 및 반복

작성된 코드는 에디터 창에서 바로 확인할 수 있어요.

- HTML/CSS/JS 프로젝트라면 파일 탐색기에서 `index.html` 우클릭 → **Open with Live Preview** 로 브라우저에서 미리보기
- 수정이 필요하면 Gemini 채팅창에 이어서 요청하기

***

**바이브 코딩 핵심 요령:**
- **에러가 나면** → 에러 메시지를 Gemini 채팅창에 붙여넣기
- **수정하고 싶으면** → "배경을 어둡게 바꿔줘", "글씨 크기 키워줘"처럼 구체적으로
- **코드 일부만 수정** → 해당 파일을 열고 `Ctrl+K` (또는 `Cmd+K`) 로 인라인 수정 요청
